# Chai Culture – Coming Soon Landing Page

A premium "Coming Soon" landing page designed for Chai Culture, inspired by royal Indian heritage and warm, elegant aesthetics.

## Live Demo
https://mohdali644.github.io/Chai-Culture/

## Tech Stack
- HTML
- CSS
- JavaScript

## Design Choices
Warm cream and deep brown tones with gold accents were used to reflect the brand’s royal Indian roots. Elegant typography and a minimal layout create a premium feel, while subtle animations enhance user engagement without overwhelming the design.
